# Task 3 – Streamlit Frontend UI Setup

## Overview
Task 3 focuses on creating the frontend interface for the AI Resume Builder using Streamlit.  
The UI includes:
- User Registration Page
- User Login Page
- Home Dashboard

These pages will later connect to the backend developed in Task 4.

## Files Included
- app.py – Main entry point for the Streamlit application.
- pages/1_registration.py – User registration interface.
- pages/2_login.py – User login interface.
- pages/3_home.py – User homepage after login.

## Output
A working Streamlit UI with navigation and form elements for user interaction.
