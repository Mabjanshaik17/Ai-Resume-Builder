# AI Resume Builder - Milestone 1

This project helps users upload resumes, extract skills, analyze strengths/weaknesses, and get job recommendations.

## How to run

1. Create virtual environment:
   python -m venv venv
   venv\Scripts\activate

2. Install dependencies:
   pip install -r requirements.txt

3. Run Streamlit:
   streamlit run frontend/pages/home.py

Project Structure:
- backend/
- frontend/
- data/
